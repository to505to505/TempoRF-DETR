# Results — Tables, Wilcoxon tests, paper cross-check

All `.txt` files in this directory are the **regenerated outputs** of the eval and
statistical-analysis scripts in `scripts/`, plus a side-by-side comparison with the
numbers reported in the thesis (`thesis/conference.pdf`).

## Files

| File | What it contains | Generated by |
|---|---|---|
| `table1_module_ablation.txt` | Table I (TempoRF-DETR ablation, in-dist/OOD AP30) + per-row paper vs repo verification | `scripts/eval_video_run.py` per run |
| `table2_video_baselines.txt` | Table II (Stenosis-DetNet / PS-STT / STQD-Det / TempoRF-DETR) + verification | `scripts/eval_baseline_{detnet,psstt}.py` + `scripts/eval_video_run.py` |
| `table3_pet_alternatives.txt` | Table III (Post-Net / Prompt / TempoRF-DETR) + verification | `scripts/eval_video_run.py` per run |
| `wilcoxon_table2_vs_baselines.txt` | Per-baseline Wilcoxon p-values vs TempoRF-DETR | `scripts/wilcoxon_table2_vs_baselines.py` |
| `wilcoxon_table1_ablation_steps.txt` | Wilcoxon p-values for progressive ablation steps (Table I rows 5→6, 6→7, etc.) | `scripts/wilcoxon_table1_ablation_steps.py` |
| `wilcoxon_per_model_bootstrap.txt` | Per-model AP30 mean + 95% cluster bootstrap CI, video-level paired | `scripts/wilcoxon_per_model_bootstrap.py` |

The TempoRF-DETR full result reported in Tables I/II/III and the headline of the
thesis abstract is produced by `rfdetr_video/runs/video_overfit_R1/best.pth`
(saved at training step 8, see `best.txt`). This is the only checkpoint that
reproduces the 0.581/0.416 headline numbers; other CRRCD-distilled runs in the
repo (e.g. `video_5_etf_consistency_distill_crrcd`) drift by a few hundredths
and are not the canonical full-model run.

python scripts/wilcoxon_per_model_bootstrap.py     > results/wilcoxon_per_model_bootstrap.txt
python scripts/wilcoxon_table1_ablation_steps.py   > results/wilcoxon_table1_ablation_steps.txt

# Tables I/II/III rows (requires data/ to be downloaded first)
python scripts/eval_video_run.py video_overfit_R1
python scripts/eval_baseline_detnet.py
